00343 /*--                                                                      --*/
00344 /*-- Let T \subseteq A be a spanning tree, and consider some node v \in V --*/
00345 /*-- \setminus {0}. There is an unique (undirected) path, denoted by      --*/
00346 /*-- P(v), defined by T from v to the root node 0. The arc in P(v), which --*/
00347 /*-- is incident to v, is called the *basic arc* of v. The other terminal --*/
00348 /*-- node u of this basic arc is called the *father node* of v.           --*/
00349 /*-- The spanning tree T is represented saving the basic arc of every     --*/
00350 /*-- node, and maintaining the order of the nodes and the depth as to     --*/
00351 /*-- T root after a Post-Visit of T.                                      --*/
00352 /*-- This order is saved in a bidirectional list written in the node      --*/          
00353 /*-- The Primal Simplex uses a different data structure as to Dual        --*/          
00354 /*-- Simplex, because Dual Simplex needs additional data like Backward Star--*/
00355 /*-- and Forward Star.                                                    --*/
00356 /*-- Then, Primal Simplex uses different data structures if it is in the  --*/
00357 /*-- linear case or in the quadratic case.                                --*/
00358 /*--------------------------------------------------------------------------*/
